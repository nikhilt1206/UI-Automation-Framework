🚀 UI Automation Framework

📌 Overview

This project is a UI Automation Framework built using Selenium WebDriver, Java, and TestNG.
It is designed to automate end-to-end testing of web applications with a focus on scalability, reusability, and maintainability.

⸻

🛠️ Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Log4j2 (Logging)
* Extent Reports (Reporting)
* Apache POI (Excel handling)
* Gson (JSON handling)
* OpenCSV (CSV handling)

⸻

✨ Key Features

* Page Object Model (POM) design pattern
* ThreadLocal WebDriver for parallel execution
* Data-driven testing (Excel, JSON, CSV)
* Retry Analyzer for failed test execution
* Extent Reports integration for detailed HTML reports
* Log4j2 logging for debugging and traceability
* TestNG Listeners for centralized reporting and logging
* Headless browser execution support
* LambdaTest cloud execution support

⸻

⚡ How to Run Tests

Run using Maven:
mvn clean test

Run using TestNG:

* Execute testng.xml from IDE

⸻

📊 Reporting

* HTML reports are generated using Extent Reports
* Logs are generated using Log4j2
